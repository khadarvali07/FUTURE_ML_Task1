# Sales Forecasting Using Machine Learning

## Project Overview

This project predicts future sales trends using historical business sales data with Machine Learning techniques.

## Objective

To help businesses forecast demand, improve inventory planning, and optimize decision making.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Workflow

1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Model Training
5. Sales Prediction
6. Data Visualization

## Model Used

Linear Regression Algorithm

## Business Benefits

* Better Inventory Management
* Cash Flow Planning
* Reduced Overstock Losses
* Better Future Sales Planning
